# 자기소개

2년간 VR·AR·XR 메타버스 솔루션 기업에서 Unity 클라이언트부터 ASP.NET Core 백엔드, Linux 서버 인프라까지 프로젝트의 전체 스택을 경험한 풀스택 개발자입니다.

디지털 트윈 프로젝트에서는 메인 개발 리드로서 요구사항 분석부터 아키텍처 설계, 구현, 납품까지 전 과정을 주도했습니다. 백엔드 API 서버를 단독으로 설계·구축하고, 클라이언트의 MVVM 구조를 설계하며, 고객사와 직접 소통하는 경험을 통해 "혼자서도, 팀에서도 돌아가는 개발"을 할 수 있는 역량을 쌓았습니다.

깔끔한 구조와 유지보수 가능한 코드를 지향합니다. 동작하는 코드에서 그치지 않고, 왜 이 구조인지 설명할 수 있는 코드를 작성하려 합니다. 새로운 기술을 빠르게 습득하고 실무에 적용하는 것을 즐기며, 사내 AI 챗봇 구축이나 GPU 서버 환경 셋업처럼 필요하면 직접 찾아서 해결하는 자세로 일합니다.
